# Deck preview baseline

These PNGs are the visual regression reference for the generated Mass deck. They
are the **actual** rasterized slides (via LibreOffice) for one fixed sample
Sunday, so color and alignment changes are caught before they reach a real Mass.

- Sample: Twelfth Sunday in Ordinary Time (green season), date `2026-06-21`.
- Generated by: `scripts/render_deck_preview.py --baseline`

## Regenerate this baseline (only when an intended design change is approved)

```bash
python3 scripts/render_deck_preview.py --baseline
```

## Compare a candidate against this baseline

```bash
# 1. Render the current code to the candidate folder
python3 scripts/render_deck_preview.py            # writes outputs/preview_slides/sample

# 2. Diff the two folders (any tool works); e.g. with ImageMagick:
for f in slide_*.png; do
  compare -metric AE "baseline/$f" "../sample/$f" "diff_$f" 2>> diffs.txt || true
done
```

Unexpected differences in colors, vertical centering, or text fitting indicate a
regression in `generators/powerpoint.py` or `generators/deck_template.py`.
